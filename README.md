# Counter & Number Generator React App
A simple and interactive React application that allows users to count numbers (both positive and negative) and generate random numbers. 
The app features a tab-based interface to easily switch between the Counter and Random Number Generator.

# Repository link
https://github.com/Mahaveer-24/Counter-Number-Generator-React

# Features
 # Counter Tab
   * Increment and decrement numbers
   * Supports both positive and negative numbers
   * Reset the counter to zero
 # Random Number Generator Tab
   * Generate random numbers within a user-defined range
   * Enter minimum and maximum values using input fields
   * Switch back and forth easily using tabs
 
 # Tab Switching
Clean and simple interface to switch between Counter and Random Number Generator

# Technologies Used
 * React.js
 * HTML & CSS
 * JavaScript (ES6+)
 * Functional Components and React Hooks
